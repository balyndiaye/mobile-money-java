## Description du projet
Cette application simule un système de paiement mobile. Elle permet de gérer des clients, des comptes et d'effectuer des opérations financières via une interface console avec une sauvegarde des données sous MySQL.

## Fonctionnalités
- Gestion des Clients (Ajout, Liste)
- Gestion des Comptes (Création, Solde)
- Opérations : Dépôt, Retrait, Transfert
- **Bonus :** Paiement Marchand & Interface colorée

## Installation
1. Cloner le dépôt : `git clone [lien-du-repo]`
2. Importer le fichier `database.sql` dans votre serveur MySQL.
3. Configurer les accès dans `DatabaseConnection.java`.
4. Lancer `Main.java`.
